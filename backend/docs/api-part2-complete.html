User Management

POST /api/users Create user

Create a new user. Requires Admin or SuperAdmin role.

Request Body

ParameterTypeRequired
usernamestringRequired
emailstringRequired
passwordstringRequired
rolestringRequired
curl -X POST http://localhost:5000/api/users \ -H "Authorization: Bearer YOUR_TOKEN" \ -H "Content-Type: application/json" \ -d '{"username": "new_user", "email": "user@example.com", "password": "pass123", "role": "editor"}'
GET /api/users List users

Get all users (filtered by role).

curl -X GET http://localhost:5000/api/users \ -H "Authorization: Bearer YOUR_TOKEN"
DELETE /api/users/:userId Delete user

Soft delete a user.

curl -X DELETE http://localhost:5000/api/users/USER_ID \ -H "Authorization: Bearer YOUR_TOKEN"

Note: This is a demo - actual deletion requires a valid user ID

Content Management

POST /api/content Create content

Create new content. Requires Editor role or above.

Request Body

ParameterTypeRequired
titlestringRequired
bodystringRequired
statusstringOptional
curl -X POST http://localhost:5000/api/content \ -H "Authorization: Bearer YOUR_TOKEN" \ -H "Content-Type: application/json" \ -d '{"title": "Test Article", "body": "Content here...", "status": "draft"}'
GET /api/content List content

Get all content items.

curl -X GET http://localhost:5000/api/content \ -H "Authorization: Bearer YOUR_TOKEN"
