Image Management

POST /api/images Upload image

Upload an image file (JPG, PNG, WebP).

curl -X POST http://localhost:5000/api/images \ -H "Authorization: Bearer YOUR_TOKEN" \ -F "image=@/path/to/image.jpg" \ -F "title=My Image"
GET /api/images List images

Get all uploaded images.

curl -X GET http://localhost:5000/api/images \ -H "Authorization: Bearer YOUR_TOKEN"

File Management

POST /api/files/upload Upload file

Upload any file (PDF, DOC, ZIP, etc.).

curl -X POST http://localhost:5000/api/files/upload \ -H "Authorization: Bearer YOUR_TOKEN" \ -F "file=@/path/to/document.pdf"
GET /api/files List files

Get all uploaded files.

curl -X GET http://localhost:5000/api/files \ -H "Authorization: Bearer YOUR_TOKEN"

Analytics

GET /api/analytics/dashboard Dashboard statistics

Get comprehensive dashboard stats. Requires Admin or SuperAdmin.

curl -X GET http://localhost:5000/api/analytics/dashboard \ -H "Authorization: Bearer YOUR_TOKEN"

Notifications

GET /api/notifications List notifications

Get all notifications. Requires Admin or SuperAdmin.

curl -X GET http://localhost:5000/api/notifications \ -H "Authorization: Bearer YOUR_TOKEN"
